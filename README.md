## ⚡ Ejemplo: 4
```diff
<!DOCTYPE html>
<html>
<head>
    <title>hola</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <style>
        h1 {
            color: red;
        }
        @media only screen and (max-width: 600px) {
            h1 {
                color: green;
            }
        }
    </style>
</head>
<body>
    <h1>¡Hola, bienvenido!</h1>
</body>
</html>
```
